# IDX Data Analyst Project

## Overview

This project analyzes real estate data using Python and Jupyter Notebook.

## Files

* `analysis.ipynb` → main analysis notebook
* Data files → used for analysis
* Note: API-related scripts are not included for security reasons

## Tools Used

* Python
* Pandas
* Jupyter Notebook (via VS Code)

## Description

Performed data cleaning, exploration, and analysis on CRMLS datasets to extract insights.
